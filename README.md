# idmc-home-miner-breakeven-calculator

IDMC Home Miner Break-Even Calculator

A browser-based tool for estimating daily/monthly profitability of home LTC/DOGE mining rigs. Calculates coin yield from live network difficulty and hashrate, compares against real-time coin prices and Texas electricity rates, and reports break-even cost per kWh.

No backend required — runs entirely client-side.
